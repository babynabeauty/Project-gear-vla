# GEAR-VLA Anonymous Project Page

This is a static double-blind project page for the CoRL submission materials.

Run it from the repository root so the page can resolve the original video assets:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000/gear-vla-site/
```

This package is prepared for GitHub Pages publication. It states that code and models will be
released after the review-compatible release point.
